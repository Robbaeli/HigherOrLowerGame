# HigherOrLowerGame
Higher or Lower - Android Game

Simple Higher or Lower card game made with Kotlin.

Guess if the next card will be higher or lower.
Correct = +1 point | Wrong = 0 points
Play through all 52 cards and try to get the highest score!
